A responsive NGO website created for She Can Foundation as part of a Frontend Development Internship Selection Task.

Project Overview

This website is designed to showcase the mission and vision of She Can Foundation, an organization dedicated to empowering women through education, opportunities, and community support.

The website includes:

Home/Hero Section
About Us Section
NGO Image
Volunteer / Join Us Button
Responsive Design
Footer Section
