# She Can Foundation - NGO Frontend Project

A simple, professional, and mobile-responsive landing page for the **She Can Foundation**—a non-profit organization dedicated to empowering women and supporting youth through education, mentorship, and community impact.

This project is built using pure **HTML5**, **CSS3**, and basic **Vanilla JavaScript** (under 60 lines), structured specifically as a college-level frontend portfolio project or internship submission task.

---

## 📁 Project Structure

```text
SheCanFoundation/
│
├── index.html         # Webpage structure & semantic sections
├── style.css          # Color variables, typography, and flexbox layouts (under 200 lines)
├── script.js          # Minimal JavaScript event handling and form validation
└── images/
    └── ngo-image.jpg  # Representative education/social impact photo
```

---

## ✨ Features

1. **Clean & Professional Theme**: Features a soft pastel purple, pink, and white color palette.
2. **Fully Responsive**: Adapts seamlessly to desktop viewports, tablets, and mobile devices using CSS Flexbox.
3. **Simple Navigation**: Standard sticky header with active logo and click navigation targets.
4. **Hero Banner**: High-impact banner with main heading, tagline, and call-to-action button.
5. **Two-Column About Grid**: Side-by-side text list and illustrative photo.
6. **Volunteer & Contact Sections**: Minimalist forms and buttons that trigger simple client-side responses.
7. **No Heavy Libraries**: Uses zero complex frameworks, React, Bootstrap, Tailwind, or large script plugins to remain lightweight and easy to understand.

---

## 🛠️ Tech Details

* **HTML**: Semantic HTML5 elements (`<header>`, `<nav>`, `<section>`, `<footer>`).
* **CSS**: Clean, organized properties, using custom root variables for colors, with a mobile breakpoint at `768px`.
* **JavaScript**: Simple DOM event listeners (`click`, `submit`), modal alerts, and smooth section transitions.

## 🚀 How to Run the Website

You can run this project instantly in two ways:

### Method 1: Double-click index.html
Open the `SheCanFoundation` folder and double-click the `index.html` file to launch it directly in Google Chrome, Microsoft Edge, Firefox, or Safari.

### Method 2: Live Server in VS Code
1. Open the project folder in **Visual Studio Code**.
2. Install the **Live Server** extension by Ritwick Dey.
3. Click the **"Go Live"** button in the status bar at the bottom-right corner.
4. The page will run locally on `http://127.0.0.1:5500`.
